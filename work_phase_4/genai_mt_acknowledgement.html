GenAI/MT Acknowledgement

This page documents my responsible use of Generative AI and Machine Translation (GenAI/MT) tools for Work Phase 4, including transparency, checklist completion, reflection, and an integrity statement.

1. Acknowledgement of GenAI/MT Use

Tools used: ChatGPT (coding assistance, accessibility guidance, copy editing), Google Translate/DeepL (spot-checking phrasing), VS Code extensions (linting hints). No AI was used to generate final deliverables end-to-end; all outputs were reviewed, edited, and verified by me.

  • Design & Front‑end support: code scaffolding, ARIA patterns, keyboard interaction logic, CSS layout suggestions.
  • Documentation support: outlining sections, grammar checks, concision edits.
  • Accessibility support: reminders to add labels, roles, focus states; test heuristics prior to IBM checker.

2. GenAI/MT Usage Checklist

Item Status Evidence
Transparency (what, why, when) Completed Documented on this page; commit messages reference AI-assisted changes.
Purposeful application aligned to task goals Completed Used for ideation, code reviews, and accessibility guidance; not for final answers without verification.
Critical engagement (evaluate/modify outputs) Completed Rewrote prompts; compared variants; tested code locally and on UQCloud.
Personal voice & authorship Completed Final copy edited by me for tone and accuracy; design decisions justified in rationale.
Reflection on learning Completed See Section 4 & 5.

3. How GenAI/MT Supported This Project

3.1 Productivity & Quality

  • Faster prototyping of calendar interactions (arrow‑key nav, ARIA roles).
  • Rapid code review suggestions (naming, modularisation, token consistency).
  • Concise wording for UI microcopy (labels, hints, alt text).

3.2 Accessibility

  • Pre‑flight checks before running IBM Equal Access Checker.
  • Contrast and focus visibility reminders; screen reader text patterns.
  • Keyboard-only paths verified (no traps, logical tab order).
