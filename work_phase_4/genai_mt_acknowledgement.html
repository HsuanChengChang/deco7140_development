GenAI/MT Acknowledgement

This page documents my responsible use of Generative AI and Machine Translation (GenAI/MT) tools for Work Phase 4, including transparency, checklist completion, reflection, and an integrity statement.

1. Acknowledgement of GenAI/MT Use

Tools used: ChatGPT (for generating initial code and content drafts), Google Translate/DeepL (for quick phrase checks), and VS Code extensions (for linting and syntax hints). I used AI tools mainly to generate drafts and prototypes which I then carefully reviewed, refined, and edited to ensure they met my project’s specific goals. Sometimes, the AI generated unnecessary or redundant code; in such cases, I removed the extra code and kept only the essential parts that aligned with my intended design and logic.

  • Design & Front-end support: Used AI to scaffold layout ideas, generate ARIA patterns, and draft basic JS logic, followed by manual review and cleanup.
  • Documentation support: Generated section outlines or initial drafts, then rewritten by me for clarity and accuracy.
  • Code refinement: Verified each AI-generated snippet for functionality, removed redundant parts, and optimised final code for readability and performance.

3. How GenAI/MT Supported This Project

3.1 Productivity & Quality

  • Faster prototyping of calendar interactions (arrow‑key nav, ARIA roles).
  • Rapid code review suggestions (naming, modularisation, token consistency).
  • Concise wording for UI microcopy (labels, hints, alt text).

3.2 Accessibility

  • Pre‑flight checks before running IBM Equal Access Checker.
  • Contrast and focus visibility reminders; screen reader text patterns.
  • Keyboard-only paths verified (no traps, logical tab order).
